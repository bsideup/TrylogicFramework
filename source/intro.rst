Intro
==========================

Basic overview
--------------------------------------

TrylogicFramework is a MVCS framework.

Main design patterns are:

#. Model-View-Controller-Service;
#. IoC with dependency injection;
#. Global Dispatching;
#. Observer;
#. Delegating.

It's provide low-linking, very efficient project structure.
